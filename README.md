---

## 🚀 Overview

This portfolio simulates a **desktop operating system inside the browser** with draggable windows, a working dock, multiple applications, and interactive UI elements.  
It highlights my ability to build **scalable and creative web applications using modern technologies.**

---

## ✨ Key Features

🖥 **Desktop Environment**
- macOS-style UI with menu bar and dock
- Smooth animations and responsive design

🪟 **Window Management**
- Draggable windows
- Minimize, maximize, and multi-window support

📱 **Built-in Apps**
- Finder (About Me)
- Safari (Projects)
- VS Code (Developer view)
- Terminal (Interactive commands)
